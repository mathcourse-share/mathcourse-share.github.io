    Make 基础

    什么是 Make

    Make 是一个自动化构建工具，使用 Makefile 文件来定义如何编译和链接程序。它通过检查文件的时间戳来决定哪些文件需要重新编译。

    Makefile 的基本结构

    Makefile 的基本结构由目标、依赖和命令组成，通常形式为：

    target: dependencies     
    command

    Make 的常用命令

    • make：执行默认目标，与make all等效。
    • make <target>：执行定义的<target>目标，如果没有这个目标将返回错误信息。
    • make -j：并行执行构建，使用本机的全部线程
